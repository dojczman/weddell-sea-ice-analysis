# Spatial Changes in Sea Ice Fragmentation and Configuration in the Weddell Sea

This repository presents a GIS-based analysis of sea ice changes in the Weddell Sea (Antarctica) between January 2019 and January 2021 using Sentinel-2 satellite imagery.

The project was completed as part of the Postgraduate Diploma in Geographic Information Systems (GIS) at the University of Gdańsk.

The workflow combines satellite image preprocessing, NDSI-based sea ice extraction, raster processing, polygonization, and spatial analysis to quantify changes in sea ice extent and fragmentation.

## Key Findings

- Sea ice extent decreased from **8,379.91 km²** to **2,669.17 km²**.
- The number of detected ice floes decreased by approximately **57%**.
- Mean ice floe size remained relatively stable, suggesting that the primary change was a reduction in the number of floes rather than their size.

## Methods

- Sentinel-2 MSI imagery
- ArcGIS Pro
- NDSI (Normalized Difference Snow Index)
- Majority Filtering
- Raster Classification
- Polygonization
- Spatial Statistics
